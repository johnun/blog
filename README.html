Questions

  1. What are we building? We are building a personal site. A place where we can blog, share examples or our work, and have people contact us.
  2. Who are we building it for? We are building it for ourselves, but also the community. Sharing what we are learning by blogging is a great way to learn for ourselves, but we teach others in the process. Show potential employers that we know what we are doing.
  3. What features do we want to have?
    • Posts
    • Create / Edit / Destroy
    • Markdown
    • Syntax highlighting
    • Comments (Disqus)
    • Projects
    • Create / Edit / Destroy
    • Contact
    • Contact form
    • Sendgrid
    • User (Devise)

User Stories

As a blank, I want to be able to blank, so that blank. - As a user, I want to be able to create posts so that I can share what I am learning on my blog. - As a user, I want to be able to edit & destroy posts, so that I can manage my blog. - As a user, I want to be able to write posts in markdown format so that it’s easy for me to write posts.
